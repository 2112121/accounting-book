# 記帳本

個人收支管理應用，使用 React + TypeScript + Firebase 構建。

## 功能

### 收支記錄
- 「記一筆」彈窗統一新增支出與收入，頂部 iOS 滑動 tab 切換
- 支出／收入各自完整分類（餐飲、交通、薪資、獎金…等）
- 多幣種輸入，自動換算為 TWD（即時匯率）
- 編輯、刪除既有記錄

### 定期費用／收入
- 設定週期規則（每日／每週／每月／每年），自動補產到期記錄
- 補產採確定性 doc ID + transaction，防止重複
- 「固定收支管理」統一管理介面，可編輯金額、停用、刪除規則

### 總覽與分析
- 頂部統計卡：本月支出、本月收入、本月結餘
- 預算進度條
- 支出分析圓餅圖（當月／選擇月份／全部），點擊分類看明細
- 收入分析圓餅圖（同樣支援篩選）
- 每日趨勢長條圖，支出／收入各有獨立 iOS 滑動切換
- 圓餅圖與每日趨勢 tab 互相獨立

### 歷史明細
- 全部／支出／收入 滑動 tab 篩選
- 多種日期範圍（今日、昨日、本週、上週、本月、自訂）
- 收入以綠色 `+` 顯示，支出以粉色 `−` 顯示

### 其他
- Firebase 帳號登入，資料隔離
- 好友系統與排行榜
- 響應式設計，支援桌面與行動裝置

## 技術棧

- **前端**: React 19 + TypeScript + Vite
- **樣式**: Tailwind CSS（搭配 inline style 處理 iOS 滑動元件）
- **圖表**: ECharts
- **後端**: Firebase Firestore + Authentication + Hosting
